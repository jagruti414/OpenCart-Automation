└── utilities/      → BaseClass

pom.xml             → Maven Dependencies

## ▶️ How to Run
1. Clone: `git clone https://github.com/jagruti414/OpenCart-Automation.git`
2. Open in Eclipse as Maven Project
3. Right click testng.xml → Run As → TestNG Suite
